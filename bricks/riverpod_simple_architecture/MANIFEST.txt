# Brick-owned file enumeration — every file `mason make` emits into the host
# app (after hooks), relative to the app root. Format: <path>[<TAB><condition>]
# Conditions: responsive (only when responsive=true),
#             per-language (one per selected language; {lang} placeholder,
#                           'en' variants are always present),
#             per-feature (one per features[] entry; {feature} placeholder),
#             per-feature-data (only for features with data=true),
#             generated (created by post_gen codegen — slang/build_runner).
# The host `flutter create` project provides everything else (android/, ios/,
# platform config); post_gen PATCHES (never creates) android/ and ios/ files:
#   android/app/build.gradle(.kts) applicationId, AndroidManifest android:label
#   + android:enableOnBackInvokedCallback, ios/Runner.xcodeproj bundle id,
#   ios/Runner/Info.plist CFBundleDisplayName.
